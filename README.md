# HireMe Frontend

Next.js frontend project for HireMe.

## Features
- Signup and login forms
- Homepage with requirement listing
- Post requirement page
- Requirement detail page
- Calls external backend API

## Setup
1. Create `.env.local`:
```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:4000
```
2. Run:
```bash
npm install
npm run dev
```

Frontend runs on `http://localhost:3000`.
